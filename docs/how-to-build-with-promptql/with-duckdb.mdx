## Tutorial

<BoilerplateInit />

Select `hasura/duckdb` (you can type to filter the list). Then, enter the following environment variables:

| ENV          | Example             | Description                                                                              |
| ------------ | ------------------- | ---------------------------------------------------------------------------------------- |
| `DUCKDB_URL` | `/path/to/*.duckdb` | The path to the DuckDB database file which will be mounted to the connector's container. |
